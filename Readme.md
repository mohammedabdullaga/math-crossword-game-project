# __User Stories:__

#### As a __player__, I want to see a math puzzle with empty cells so I can fill them in

#### As a __player__, I want to see available numbers that get used when I type them so i know what numbers are left in my option

#### As a __player__, I want to know if i missed any cell without a number

#### As a __player__, I want to check if my solution is correct after i finished the puzzle

#### As a __player__, I want to play the game again if I loose



 ## ========================================



# __Pseudo Code:__

#### __WHEN page loads:__
 - CREATE puzzle grid with empty inputs
  <br>
 - SHOW available numbers



__WHEN user inserting all the numbers:__
<br>
  - IF all inputs are filled:
  <br>
    CHECK result button are pressed and show the massage of wining or lose
    <br>
  - ELSE:
  <br>
    SHOW "Please, fill all cells!" message

__WHEN user clicks CLEAR button:__
<br>
  - CLEAR all inputs
  <br>
  - RESET all numbers to be available again
